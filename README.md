## Setup
#### [Link to live](https://competent-panini-c57d95.netlify.app/) <br/>

Clone down this repository. You will need `node` and `npm` installed globally on your machine.

Installation:
`npm install`

To Start Server:
`npm start`

To Visit App:
`localhost:3000`
